# Codex Account Switcher 1.8.3.2

Version 1.8.3.2 improves multi-account usage accuracy and gives the account panel a softer, more readable frosted appearance.

## Live usage for every account

- Refresh now reads live five-hour and weekly limits for every saved account, not only the active account.
- Account requests run concurrently so a complete refresh remains quick.
- A short cache prevents unnecessary repeat requests when the panel is reopened.
- If one account temporarily fails to refresh, its last reliable usage remains visible instead of being replaced with stale or invented data.
- Opening the account panel triggers a fresh all-account update, and the refresh controls now describe that behaviour accurately.

## Softer frosted interface

- The outer clear Liquid Glass effect has been replaced by the standard macOS popover material for a calmer frosted background.
- Account cards, settings sections, buttons, and usage information remain solid and readable.
- The existing graphite colour system, active-account emphasis, compact grid, and navigation are unchanged.

## Maintenance

- Large UI and model definitions have been separated from the main application file to keep future changes safer and easier to review.
- Infrastructure coverage now includes refresh caching and last-known-good snapshot retention, bringing the automated suite to 21 assertions.
- The app now targets the current `/Applications/ChatGPT.app` desktop surface directly.

The release is ad-hoc signed and includes a SHA-256 checksum. It is not Apple-notarized, so macOS may show the standard warning for independently distributed software.
